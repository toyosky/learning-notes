# 多模态大模型入门 - 从 LLM 到 VLM

## VLM 发展时间线

```
2021.01  CLIP（OpenAI） — 图文对齐的里程碑
2021.06  ViT（Dosovitskiy） — 用 Transformer 做视觉
2022.04  Flamingo（DeepMind） — 少样本多模态
2023.04  LLaVA（UWisc） — 简易架构，开源标杆
2023.09  Qwen-VL（阿里） — 中文多模态
2024.02  LLaVA-NeXT — 动态分辨率改进
2024.05  GPT-4o — 全模态实时交互
2025.01  Qwen2.5-VL — 更强的中文 VLM
```

## 两个关键阶段

### 阶段一：预训练（Pre-training）

目标：学习通用知识，打基础。

- **LLM**：读完互联网上的文字 → 学会语言规律
- **CLIP**：看几亿张图片 + 文字描述 → 学会"图和文说的是同一件事"
- **ViT**：学会把图片切成小块，理解视觉特征

### 阶段二：监督微调（SFT）

目标：学会"对话"，能听懂指令。

- 用人工标注的"指令 → 回答"对来训练
- 比如：输入"这张图里有什么？" → 训练模型输出描述

## 单模态 vs 多模态

| | 输入 | 输出 | 代表模型 |
|--|------|------|---------|
| **LLM** | 文字 | 文字 | GPT-4, Qwen, DeepSeek |
| **VLM** | 图片+文字 | 文字 | Qwen-VL, GPT-4V, LLaVA |
| **全模态** | 任意 | 任意 | GPT-4o |

VLM 的关键突破：**模态对齐** — 让模型理解"图片和文字说的是同一件事"。

## CLIP 的核心思想

```
图片 → CLIP 图片编码器 → [0.1, 0.3, -0.2, ...]
"猫" → CLIP 文字编码器 → [0.1, 0.3, -0.2, ...]  ← 向量接近！
```

对比学习训练：同一个概念的图文向量距离近，不相关概念的向量距离远。

## VLM 架构（主流方案）

```
图片 → 视觉编码器(ViT/CLIP) → 视觉特征 → 投影层(MLP) → LLM → 回答
                                                          ↑
用户问题 → 文本分词 → 嵌入 → ————————————————
```

### 关键组件

| 组件 | 作用 | 常见选择 |
|------|------|---------|
| **视觉编码器** | 把图片转为特征向量 | SigLIP, CLIP-ViT, InternViT |
| **投影层** | 视觉特征 → LLM 词表空间 | 单层 Linear 或 2 层 MLP |
| **LLM 基座** | 理解融合信息并生成回答 | Qwen, LLaMA, DeepSeek |

### 主流 VLM 对比

| 模型 | 视觉编码器 | LLM 基座 | 特点 |
|------|-----------|---------|------|
| **LLaVA-1.5** | CLIP-ViT-L | LLaMA-2 | 简单有效，开源标杆 |
| **Qwen-VL** | CLIP-ViT-bigG | Qwen-7B | 中文支持好，多图输入 |
| **Qwen2.5-VL** | SigLIP | Qwen2.5 | 动态分辨率，视频理解 |
| **GPT-4V/4o** | 未公开 | GPT-4 | 综合能力最强 |

## VLM 的典型能力

1. **视觉问答** — "这张图里有什么？"
2. **文档理解** — "这张发票的总金额是多少？"
3. **场景推理** — "这个驾驶场景是否安全？"
4. **多图对比** — "这两张图有什么不同？"
5. **视频理解** — 逐帧分析视频内容
