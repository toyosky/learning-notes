# 组合分配方法：总览

> **核心问题**：权重怎么分比 1/3 更好？

本文是系列笔记的总览，介绍三种分配方法的思想，并给出选型指南。

---

## 三种分配思想

| 方法 | 核心逻辑 | 一句话 |
|------|---------|--------|
| **等权** | 1/n | 简单、稳健 |
| **风险平价** | 波动率倒数 | 低波多配、高波少配 |
| **动量排名** | 动量 ÷ 波动率 (ram) | 选涨得稳的买 |

### 何时选择哪种方法

```
你的目标是什么？
│
├─ 最大程度简化 → 等权
│
├─ 稳定的风险控制，不在乎收益高低
│   → 风险平价（配的逻辑）
│
├─ 追求收益，能接受集中持仓风险
│   → 动量排名（选的逻辑）
│
└─ 结合两者？
    → 风险平价作底仓 + 动量排名作战术调整
```

### 三种方法的本质区别

```
风险平价:
  永远满仓，权重随波动率缓慢变化
  → "配"的逻辑：低波多配、高波少配
  → 持仓稳定，调仓温和

动量排名:
  不满仓，ram 为负的剔除，ram 为正的按分数配
  → "选"的逻辑：只买涨得好的
  → 持仓集中，调仓剧烈

等权:
  永远 1/n，不管市场怎么变
  → "躺"的逻辑：没有观点就是最好的观点
  → 对初学者最友好，回测最不易过拟合
```

## 快速对比总结

| 维度 | 等权 | 风险平价 | 动量排名 |
|------|------|---------|---------|
| **收益** | 🟡 中 | 🟢 高 | 🔥 最高 |
| **风险控制** | 🟡 中 | 🟢 最好 | 🟡 中等（集中度高） |
| **夏普比率** | 🟡 中 | 🟢 高 | 🟢 最高 |
| **回撤控制** | 🟡 -17% | 🟢 -12% | 🔥 -10% |
| **调仓频率** | 🟢 低 | 🟡 中 | 🔴 高 |
| **理解难度** | 🟢 简单 | 🟡 中等 | 🟡 中等 |
| **行为风险** | 🟢 容易坚持 | 🟢 容易坚持 | 🔴 大幅切换考验心态 |

> **夏普最高不代表最安全**。动量排名在 2024 年 100% 持有纳指100的行为风险极高——如果纳指回调赶上切换不及时，回撤可能远超历史水平。

## 后续方向

- 均值-方差（Markowitz）优化 → 引入收益率预期
- 风险预算（Risk Budgeting）→ 手工指定各资产的风险占比
- 动态止损阈值（基于波动率/夏普调整，而非固定百分比）
- 分批止损（部分减仓而非全仓清仓）
- 实盘模拟（考虑滑点 + 延迟成交）
