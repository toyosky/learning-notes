# 前复权 vs 后复权 — 选错直接算错收益率

## 一句话

**复权** 是为了消除分红、送股、拆股等公司行为造成的价格"虚假跳变"，让历史价格在同一个基准上可比。
**前复权** 调低历史价格，**后复权** 调高最新价格。

---

## 为什么需要复权？

股票/ETF 的价格会因公司行为产生"非交易性跳变"：

| 事件 | 价格影响 | 例子 |
|------|---------|------|
| **现金分红** | 除息日开盘价 = 前收盘 - 每股分红 | 股价从 10 元 → 9.5 元（分红 0.5 元） |
| **送股/转增** | 除权日开盘价按比例下调 | 10 送 10 → 股价直接腰斩 |
| **拆股 (split)** | 价格按比例下调 | 1 拆 5 → 股价降到 1/5 |
| **合股 (reverse split)** | 价格按比例上调 | 10 合 1 → 股价变 10 倍 |

**如果不复权**，K 线图上会出现一个"虚假缺口"：

```
价格
│
│  10 ──┤ 分红 0.5 元
│  9.5 ─┤ ← 这不是亏损，是钱已经分到你账上了
│
└────────────────── 时间
```

这个"下跌"不是市场行为，是**会计调整**。计算收益率时必须把它消除。

---

## 前复权（Forward Adjustment）

### 定义

**调低历史价格**，让所有价格和当前价格保持连续。当前的最新价就是实际成交价。

### 公式

```
前复权价格 = 原始价格 × 复权因子

复权因子 = (除权前价格 - 分红) / 除权前价格
         = 除权后价格 / 除权前价格
```

多次除权时，因子**连乘**：

```
P_adjusted(t) = P_raw(t) × Π factor_i  (对所有发生在 t 之后的除权事件 i)
```

### 例子

```
2021-01-01 收盘 10 元
2021-06-30 分红 0.5 元（除权后开盘 9.5 元）
          → 复权因子 = 9.5 / 10 = 0.95
2022-01-01 前复权价格 = 10 × 0.95 = 9.5 元（历史被调低了）
```

### 用代码算

```python
import pandas as pd

def forward_adjust(df: pd.DataFrame, dividend: float, ex_date: str):
    """
    df: 原始日线数据，列含 'close'
    dividend: 每股分红
    ex_date: 除权日（格式 YYYY-MM-DD）
    """
    factor = 1.0
    for date in sorted(df.index):
        if date < pd.Timestamp(ex_date):
            # 除权之前的价格，都需要调低
            factor *= (df.loc[ex_date, 'close'] / df.loc[ex_date, 'close_pre'])
    # 简化：单次除权时直接乘因子
    pre_price = df.shift(1).loc[ex_date, 'close']  # 除权前收盘
    post_price = df.loc[ex_date, 'close']           # 除权开盘
    factor = post_price / pre_price
    
    df_adj = df.copy()
    df_adj.loc[df.index < ex_date, 'close'] *= factor
    return df_adj
```

### 特点

- ✅ **最新价就是真实市价** — 实盘参考方便
- ✅ **收益率曲线连续** — 回测走势图看起来自然
- ❌ **历史价格被改变** — 不能说"历史上某天股价是 X 元"
- ✅ **量化回测最常用** — 因为回测都是从"现在"往"过去"看

## 后复权（Backward Adjustment）

### 定义

**调高历史价格**，让所有价格以首发价格为基准。历史价格不变，最新价被抬高。

### 公式

```
后复权价格 = 原始价格 / 复权因子

复权因子 = 除权后价格 / 除权前价格
```

### 例子

```
同上例：
2022-01-01 后复权价格 = 10 / 0.95 = 10.53 元
2022-01-01 实际价格 = 9.5 元
```

### 特点

- ✅ **历史价格真实** — 可以看到"历史上某天真实成交价"
- ✅ **总收益直观** — 后复权最高点 = 历史最高价值
- ❌ **最新价不是市价** — 不能直接参考当前价格
- ✅ **适合长周期分析** — 比如看贵州茅台上市以来到底涨了多少

## 代码对比：akshare 实战

```python
import akshare as ak
import pandas as pd
import matplotlib.pyplot as plt

# 获取 510300 数据：不复权 vs 前复权
raw = ak.fund_etf_hist_em(
    symbol="510300", period="daily",
    start_date="20220101", end_date="20220301",
    adjust=""       # 不复权
)

qfq = ak.fund_etf_hist_em(
    symbol="510300", period="daily",
    start_date="20220101", end_date="20220301",
    adjust="qfq"    # 前复权
)

raw_close = raw.set_index("日期")["收盘"]
qfq_close = qfq.set_index("日期")["收盘"]

# 对比：分红日前后
comparison = pd.DataFrame({
    "不复权": raw_close,
    "前复权": qfq_close,
})
print(comparison.head(10))

# 收益率差异
ret_raw = (raw_close.iloc[-1] / raw_close.iloc[0] - 1) * 100
ret_qfq = (qfq_close.iloc[-1] / qfq_close.iloc[0] - 1) * 100
print(f"\n不复权收益率:  {ret_raw:.2f}%")
print(f"前复权收益率:  {ret_qfq:.2f}%")
```

预期输出：

```
           不复权  前复权
日期
2022-01-04  5.062  5.062  ← 开始日相同
2022-01-05  5.152  5.152
...         ...    ...
2022-01-17  ✓分红除权日出现差异
2022-01-17  4.901  4.901  ← 除权后，两者重新对齐
            ↑不符   ↑连续
```

**关键差异**：分红除权那天，不复权价格直接跳空向下；前复权把除权前的价格也调低了，所以曲线是连续的。

## 对量化交易的影响

### 1. 收益率算不准

| 场景 | 不复权 | 前复权 | 后复权 |
|------|-------|-------|-------|
| 短期回测（无分红） | ✅ 正确 | ✅ 正确 | ✅ 正确 |
| 长期回测（有分红） | ❌ **偏低** | ✅ 正确 | ✅ 正确 |
| 分红再投资计算 | ❌ 错误 | ✅ 近似 | ✅ 精确 |

**具体偏差**：以沪深300ETF（510300）为例，2021-2025 年累计分红约 0.5 元/份。
不复权计算 5 年收益率 ≈ **-10%**，前复权计算 ≈ **+5%**（含分红再投资效果）。

### 2. 技术指标失真

**均线**：
- 不复权：分红日前后均线可能被"虚假跳空"拉偏
- 前复权/后复权：均线连续，信号可靠

**MACD / RSI 等**：
- 对价格跳变敏感的指标，需要复权数据
- 不复权可能在分红日出现错误金叉/死叉信号

### 3. 策略回测偏差矩阵

| 策略类型 | 对复权敏感性 | 建议 |
|---------|-------------|------|
| **高频 / 短线**（持有 < 5 天） | 🟢 低 | 不复权或前复权均可 |
| **趋势跟踪 / 均线策略** | 🟡 中 | **必须前复权** |
| **定投 / DCA** | 🔴 高 | **必须复权**，否则收益率低估 |
| **跨资产比价** | 🔴 高 | 统一复权方式 |
| **期权 / 衍生品** | 🔴 高 | 需原始价格 + 复权因子分开处理 |

### 4. 实盘注意

前复权回测假设：**分红可以按除权后的价格再投资**。但实际中：
- 现金分红部分会扣税（A 股股息税 ≈ 持有期不同，税率 0-20%）
- 红利再投资需要按实际到手金额计算
- 前复权忽略了税费，会高估实际收益

如果需要精确到税费级别，需要**自己处理除权除息事件**，而不是简单用前复权。

## 什么时候用哪个

| 你要干什么 | 选哪个 |
|-----------|-------|
| 看当前价格 | 不复权（或前复权，最新价相同） |
| 技术分析（均线、MACD） | **前复权** |
| 长期回测收益率 | **前复权** |
| 计算总收益倍数（上市以来涨了多少倍） | **后复权** |
| 观察真实历史成交价 | 不复权 |
| 实盘策略开发 | 前复权回测 + 手工扣除税费 |
| 跨资产收益比较 | 前复权（起点统一） |

## akshare 参数速查

| adjust 参数 | 含义 | 等价于 |
|------------|------|--------|
| `adjust=""` | 不复权 | 原始成交价 |
| `adjust="qfq"` | **前复权** ✅ 推荐 | 调低历史价格 |
| `adjust="hfq"` | 后复权 | 调高当前价格 |
