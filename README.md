## 使用方式

### 在 GitHub 上浏览
直接在线查看 `.md` 文件即可，wiki 链接暂不支持跨文件跳转。

### 在 Obsidian 中打开（推荐）
1. `git clone` 到本地
2. Obsidian → 打开本地仓库 → 选择此目录
3. 享受双向链接和图谱视图（Graph View）
